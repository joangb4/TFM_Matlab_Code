# TFM_Matlab_Code
Repositorio que contiene el código MATLAB desarrollado para llevar a cabo el trabajo final de máster: Diseño e implementación de un controlador predictivo no lineal en una planta piloto de procesos químicos.
El repositorio cuenta con funciones de construcción de controlador, observador y modelo; rutina de implementación en la aplicación de LabVIEW; y funciones de verificación y validación.
